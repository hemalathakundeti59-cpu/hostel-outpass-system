# Hostel Outpass System

A full-stack application to manage hostel outpass requests for students.

## Features
- Apply for outpass
- View all outpasses
- Approve/Reject/Delete requests (Admin)

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## How to Run
1. Clone the repo
2. `npm install` in both frontend & backend
3. Set up `.env` for MongoDB
4. `npm run dev` in backend
5. `npm start` in frontend (or `npm run build` for production)
